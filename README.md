indUserAgentInfo
==================

A plugin for LimeSurvey to use dropdown in replace of input in array text

## Installation

### Via GIT
- Go to your LimeSurvey Directory (version up to 2.06 only)
- Clone with Submodule in plugins/arrayTextAdapt directory
